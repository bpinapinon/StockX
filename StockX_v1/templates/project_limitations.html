Recognizing the Limitations of Our Analysis

Data itself is a human creation, so it's no surprise that there are limits to its usability. Extraction and manipulation to extract meaningful results is only the first step. One must consider not just the "how" but also "what" you use the data for. The limiations that come along with those decisions must not be overlooked once results are generated.

Limitations

Data Size

We only scrubbed for the 50 "Top Seller" and 50 "Most Popular" Nike & Adidas shoes. Things to consider with small datasets include:

  • Outliers in small sample can significantly alter the analysis.
  • You don't have enough samples to generate "train" and "sample" data.
  • Missing values for a record could have considerable impact on results.



Static Data

We scrubbed the data on a specific date. The shoes that were sitting at the "top" of the results with the criteria we selected so we're working with a static data set. For our analysis to be useful by end users long term would require a time-series model. Automating a daily scrubbing or API call to StockX to fetch daily data sets would keep our model up-to-date.



Macroeconomic Variables

The state of the economy on the macro level will have an impact on the demand of luxury items. We've been in a steady bull market, consumers have more disposable income to make these luxury purchases.

Data Variable Inconsistency

One of the critical variables in our analysis was shoe color. Color schemes included "eclectic" custom-named colors. Normalizing this variable to a known color schemes was/is a very manual process that requires human analysis.
